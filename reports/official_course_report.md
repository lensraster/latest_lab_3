# Звіт про виконання навчального проєкту

## 1. Загальна інформація

- **Навчальна дисципліна:** Основи обробки природної мови  
- **Тема завдання:** Донавчання варіанту BERT для задачі класифікації тексту  
- **Формат роботи:** практичний навчальний проєкт із повним циклом запуску, перевірки та оцінювання  
- **Репозиторій проєкту:** [https://github.com/lensraster/latest_lab_3](https://github.com/lensraster/latest_lab_3)

## 2. Мета та критерії виконання

Мета роботи полягала у створенні робочого застосунку для класифікації текстів на основі BERT із:

- вибором і донавчанням моделі;
- автоматизованою перевіркою через тести;
- документованим процесом запуску;
- звітом із фактичними результатами.

Критерієм успішності вважалося коректне завершення тестів, запуск навчання без помилок, збереження артефактів моделі та отримання передбачень на новому тексті.

## 3. Реалізоване рішення

У проєкті реалізовано:

- **Базову модель:** `bert-base-uncased` (основний експеримент);
- **Додаткову модель:** `distilbert-base-uncased` (порівняльний експеримент);
- **Датасет:** `ag_news` (класи: World, Sports, Business, Sci/Tech);
- **CLI-інтерфейс:** команди `train` та `predict`;
- **Автоматизовані тести:** `pytest`;
- **Збереження артефактів:** модель, токенайзер, метрики запуску.

Ключові файли:

- `src/nlp_bert_classifier/pipeline.py` — підготовка даних, токенізація, тренування, оцінка, інференс;
- `src/nlp_bert_classifier/cli.py` — команди запуску;
- `tests/` — набір тестів;
- `README.md` — інструкція користувача;
- `reports/run_report.md` — технічний звіт із командами та сирими результатами.

## 4. Процес запуску

### 4.1 Підготовка середовища

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
pip install -e .
```

### 4.2 Запуск тестів

```bash
pytest -q
```

Фактичний результат: **8 passed**.

### 4.3 Основний експеримент (BERT)

```bash
python -m nlp_bert_classifier.cli train \
  --train-samples 200 \
  --test-samples 80 \
  --epochs 1 \
  --batch-size 8
```

Артефакти збережено в `artifacts/bert-ag-news/`.

### 4.4 Перевірка роботи інференсу

```bash
python -m nlp_bert_classifier.cli predict \
  --model-dir artifacts/bert-ag-news/model \
  --text "Apple releases a new AI chip for data centers"
```

Отримано коректну відповідь у форматі JSON із класом і рівнем впевненості.

### 4.5 Додатковий експеримент (DistilBERT)

```bash
python -m nlp_bert_classifier.cli train \
  --model-name distilbert-base-uncased \
  --output-dir artifacts/distilbert-ag-news \
  --train-samples 200 \
  --test-samples 80 \
  --epochs 1 \
  --batch-size 8
```

Артефакти збережено в `artifacts/distilbert-ag-news/`.

## 5. Результати експериментів

- **Експеримент 1 (`bert-base-uncased`)**
  - `eval_accuracy = 0.6000`
  - `eval_f1_weighted = 0.5798`
  - `eval_loss = 1.1470`

- **Експеримент 2 (`distilbert-base-uncased`)**
  - `eval_accuracy = 0.2750`
  - `eval_f1_weighted = 0.1717`
  - `eval_loss = 1.3238`

За обраних умов (невелика навчальна підвибірка і 1 епоха) кращу якість продемонструвала базова модель `bert-base-uncased`.
